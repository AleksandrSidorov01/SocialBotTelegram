# Social Pet Bot (Tamagochi for Telegram Groups)

Интерактивный бот-питомец для групповых чатов в Telegram. Участники группы вместе заботятся о виртуальном питомце, который эволюционирует в зависимости от характера общения в чате.

## Особенности

- **Органичное взаимодействие**: Питомец живет в чате, но не спамит
- **Система показателей**: Голод, Настроение, Энергия
- **Эволюция**: От яйца до древнего существа с ветвлением по типам
- **Мини-игры**: Казино, случайные события
- **Групповая ответственность**: Общее здоровье питомца на весь чат
- **Статистика**: Таблицы лидеров и логи событий

## Требования

- Python 3.10+
- Telegram Bot Token (получить у @BotFather)

**Всё!** База данных SQLite создается автоматически, ничего не нужно устанавливать.

## Установка

1. Клонируйте репозиторий:
```bash
git clone <your-repo-url>
cd Social-Pet-Bot
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Скопируйте `.env.example` в `.env` и добавьте токен бота:
```bash
cp .env.example .env  # Linux/Mac
copy .env.example .env  # Windows
```

5. Запустите бота:
```bash
python main.py
```

База данных создастся автоматически в файле `tamagochi.db`.

## Команды бота

- `/start` - Начать взаимодействие (создать питомца в чате)
- `/status` - Показать состояние питомца
- `/feed` - Покормить питомца
- `/play` - Поиграть с питомцем
- `/gamble` - Казино на еду
- `/leaderboard` - Таблица лидеров
- `/history` - История последних событий

```
│   ├── evolution.py    # Система эволюции
│   ├── scheduler.py    # Фоновые задачи
│   └── events.py       # Случайные события
├── config.py           # Конфигурация
├── main.py             # Точка входа
├── requirements.txt    # Зависимости
└── tamagochi.db        # База данных SQLite (создается автоматически)
```
